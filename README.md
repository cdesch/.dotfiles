# dotfiles

## ZSH

[oh-my-zsh github](https://github.com/ohmyzsh/ohmyzsh)

### Setup

Link zsh aliases from dotfiles

    ln -s ~/.dotfiles/zsh/.zshrc ~/.zshrc
    ln -s ~/.dotfiles/zsh/.zshrc ~/.zshrc
    ln -s ~/.dotfiles/zsh/custom/aliases.zsh ~/.oh-my-zsh/custom/aliases.zsh



### ZSH Commands

Reload Shell
    
    source ~/.zshrc
